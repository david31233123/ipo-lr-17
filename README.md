# Магазин настольных игр "Игровой Мир"

Django-приложение для магазина настольных игр. Учебный проект, созданный в рамках лабораторной работы по интернет-программированию.

## Описание

Приложение представляет собой веб-сайт магазина настольных игр с полной системой управления товарами, категориями, производителями и корзиной пользователя.

## Технологии

- **Python** — язык программирования
- **Django 6.0.3** — веб-фреймворк
- **SQLite** — база данных (по умолчанию)
- **Pillow** — библиотека для работы с изображениями

## Модели данных

### Категория
- `название` — CharField (100 символов)
- `описание` — TextField (необязательное)

### Производитель
- `название` — CharField (100 символов)
- `страна` — CharField (100 символов)
- `описание` — TextField (необязательное)

### Товар
- `название` — CharField (200 символов)
- `описание` — TextField
- `фото_товара` — ImageField
- `цена` — DecimalField (10, 2)
- `количество_на_складе` — IntegerField
- `категория` — ForeignKey → Категория
- `производитель` — ForeignKey → Производитель

### Корзина
- `пользователь` — OneToOneField → User
- `дата_создания` — DateTimeField
- `общая_стоимость()` — метод вычисления суммы

### Элемент корзины
- `корзина` — ForeignKey → Корзина
- `товар` — ForeignKey → Товар
- `количество` — PositiveIntegerField
- `стоимость_элемента()` — метод вычисления стоимости

## Установка и запуск

1. Установите зависимости:
   ```bash
   pip install django Pillow
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd boardgame_store
   ```

3. Примените миграции:
   ```bash
   python manage.py migrate
   ```

4. Заполните базу данных (опционально):
   ```bash
   python populate_db.py
   ```

5. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

6. Откройте браузер: `http://127.0.0.1:8000/`
