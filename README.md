Магазин настольных игр "Игровой Мир"

Django-приложение для магазина настольных игр. Учебный проект, созданный в рамках лабораторной работы по интернет-программированию.

Описание

Приложение представляет собой веб-сайт магазина настольных игр с полной системой управления товарами, категориями, производителями и корзиной пользователя.

Технологии

- **Python** — язык программирования
- **Django 6.0.3** — веб-фреймворк
- **SQLite** — база данных (по умолчанию)
- **Pillow** — библиотека для работы с изображениями

Структура проекта

```
boardgame_store/
├── boardgame_store/      # Основной проект Django
│   ├── settings.py       # Настройки проекта
│   ├── urls.py           # Корневой URL-маршрутизатор
│   └── wsgi.py           # WSGI-конфигурация
├── shop/                 # Приложение магазина
│   ├── models.py         # Модели данных
│   ├── views.py          # Обработчики запросов
│   ├── urls.py           # URL-маршруты приложения
│   └── admin.py          # Настройка админ-панели
├── populate_db.py        # Скрипт заполнения БД
├── manage.py             # Утилита управления проектом
└── README.md             # Этот файл
```
